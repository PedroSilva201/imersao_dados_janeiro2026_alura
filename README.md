### Apresentação do Projeto

Este projeto foi realizado durante a imersão Dados com Python da Alura onde o certame foi usado como forma de desenvolver habillidades dentro do seguimento de ciencia de dados, o objetivo principal do mesmo era aplicar os conehcimentos vistos no Google Colab e aplicar em uma interface web usando o Streamilit.

## Metódologia

Para este cuso foi usado uma planilha CSV com os dados de empregabilidade dentro da area de dados onde foram usados para diversas finalidades seguindo o que foi planejado no escopo
