# Imersão Dados Janeiro de 2026 Alura

## Apresentação do Projeto

Este projeto foi realizado durante a imersão Dados com Python da Alura onde o certame foi usado como forma de desenvolver habilidades dentro do seguimento de ciência de dados, o objetivo principal do mesmo era aplicar os conehcimentos vistos no Google Colab e aplicar em uma interface web usando o Streamilit.

## Metódologia

Para este cuso foi usado uma planilha CSV com os dados de empregabilidade dentro da área de dados onde foram usados para diversas finalidades seguindo o que foi planejado no escopo inicial.

O projeto poderá ser visualizado aqui dentro <a href="https://imersao-dados-janeiro2026-alura-pedro.streamlit.app/"> deste link</a> para mais detalhes.

## Explicando o Streamlit

O Streamlit é uma plataforma de hospedagens de projetos semelhante ao github pages onde o certame foi usado para subir o projeto e demonstrar o sobre como eu aprendi de acordo com os aprendizados durante a imersão, um fato curioso é que o mesmo brinca como se ele estivesse fazendo alguma comida para voce.
Por outro lado a plataforma possui uma limitação onde se um determinado projeto não possui entrada de pessoas no site em 24 horas, o mesmo fica no modo de sono até quando alguem entrar e onde ele faz todo o preparo para poder inicar o sistema e poder rodar.
